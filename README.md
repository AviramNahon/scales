# scales.py

A small Python script that can display scales and chords for stringed instruments.

## Available scales and chords

| Scales | Modes | Chords |
|--------|-------|--------|
| Chromatic | Ionian | Major |
| Major | Dorian | Minor |
| Minor Natural | Phrygian | 7th |
| Minor Harmonic | Lydian | Major 7th |
| Major Pentatonic | Mixolydian | Minor 7th |
| Minor Pentatonic | Aeolian | 6th |
| Blues Pentatonic | Locrian | Augmented |
| | | Diminished |
                    
## Usage

```
scales.py [-h] [--chord] [--tuning TUNING] key scale

positional arguments:
  key                   The key
  scale                 The scale

optional arguments:
  -h, --help            show this help message and exit
  --chord, -c           Display chord
  --tuning TUNING, -t TUNING
                        String tuning
```
